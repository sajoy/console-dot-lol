To use Console.lol in your project, you can:

  • Download and include the code
  • Link to a CDN

Option 1

Download and include

1. Save the file from Github

  • Go to github.com/sajoy/console-dot-lol/blob/master/console.js
  • Click the button that says "Raw", it will take you to a page with the code as plain text
  • Right click and "Save as" or highlight and copy the entire page.
  • Save the file as "console.js" in your project's folder.

2. Link to the file from your HTML

  • Add a script tag where you prefer them in your HTML.
  • Set the 
    src
    attribute as the path to the file.

3. Check that it's working

  • Open your project in a browser.
  • Open the browser's console.
    • You can use a hot key like 
      CTRL + SHIFT + J
      in Chrome
    • or right click and select "Inspect", then change the panel to the Console
  • In the console, type 
    console.lol();
  • You should see a random laughing emoji!
